# Amazon Clone (Mini Project)

## Description
A responsive Amazon-like homepage built using HTML and CSS.  
The project focuses on layout design, responsiveness, and UI structure.

## Features
- Responsive navigation bar
- Mobile-friendly layout
- Hero section
- Product grid using CSS Grid
- Flexbox-based panel menu

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
